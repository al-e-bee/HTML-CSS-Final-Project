# Personal Portfolio Landing Page

### _From Figma to Functional Code_

---

## Project Overview

This project is a high-fidelity translation of a professional Figma design into a fully responsive personal landing page. The goal was to demonstrate the ability to take static design requirements and implement them using modern web development practices—focusing on clean code, brand consistency, and mobile accessibility.

## Tech Stack

- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom styling using **Flexbox** and **CSS Grid** for complex layouts.
- **JavaScript:** Implementation of a functional mobile navigation toggle.
- **Font Awesome:** Scalable vector icons for social links and UI elements.
- **Google Fonts:** Integration of _Inter_ and _Merriweather_ for a premium typography feel.

---

## Key Features & Technical Wins

- **Responsive Container Logic:** Implemented a global `.container` class to manage alignment across all screen sizes, ensuring content stays centered and readable without fixed-width breaks.
- **SVG Optimization:** Custom-coded SVG manipulations to match brand colors while maintaining image sharpness across high-density displays.
- **Mobile-First Approach:** Developed a custom mobile navigation menu using JavaScript and CSS Media Queries to handle smaller viewports effectively.
- **Layered Visuals:** Replicated complex Figma "image stacking" techniques using absolute positioning and z-index management.
- **Interactive UI:** Added smooth transitions and hover scales to navigation links and call-to-action buttons for an engaging user experience.

---

## Design Highlights

- **Hero Section:** A bold introduction featuring high-quality profile imagery and a direct, high-contrast Call-to-Action.
- **Collaborations:** A clean logo grid displaying professional partnerships, optimized with CSS Grid for a 2-column layout on mobile.
- **Brand Identity:** Rebranded with a custom color palette (Deep Purple & Vibrant Orange) to showcase a unique professional identity.
